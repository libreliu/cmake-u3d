BOOL IFXAuthorMaterial::m_uNormals
 

Set to TRUE if any of the faces in this material use normals, FALSE if no faces use normals.

U32 IFXAuthorMaterial::m_uNumTextureLayers

U32 IFXAuthorMaterial::m_uOriginalMaterialID
 

If the array of IFXAuthorMaterials gets smaller because the scrubber removed unused materials and repacked them, this field gives the original array index of the material record.

BOOL IFXAuthorMaterial::m_uSpecularColors
 

True if this material specifies specular vertex colors.

U32 IFXAuthorMaterial::m_uTexCoordDimensions[IFX_MAX_TEXUNITS]
 

This array gives the dimensions of the texture coordinates used for each texture layer in the material.

Texture coordinates can range from 1-dimensional to 4-dimensional.
