Detailed Description

This interface is used to open and close files remotely thru Internet.

Note:
The associated IID is named IID_IFXInet.

virtual IFXRESULT IFXAPI IFXInet::SetConnectTimeout (  const U32  connectTimeout  )  [pure virtual]
 

Sets a value that contains the time-out value to use for Internet connection requests.

Units are in milliseconds. If a connection request takes longer than this time-out value, the request is canceled. When attempting to connect to multiple IP addresses for a single host (a multihome host), the timeout limit is cumulative for all of the IP addresses.

Implemented in CIFXInternetReadBufferX.
