Detailed Description

This changes the resolution of a single IFXMesh.

If an instance of this class is being used by an IFXCLODManager then you should not change it's resolution here since you may throw off the synchronization provided by the CLODManager.

If your mesh has only one material then you should use this class to change it's resolution directly since it is faster and smaller than IFXCLODManager.
