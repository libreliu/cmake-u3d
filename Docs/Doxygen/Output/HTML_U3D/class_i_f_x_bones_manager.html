Detailed Description

An interface to IFXBonesManager_Impl, representing a subset of the bones-based animation.

This is an (almost) headerless interface to IFXBonesManager_Impl, representing a subset of the bones-based animation.

You should not delete a IFXBonesManager directly. It may seem like it works, but destructors will not be called and you will leak memory. You must use ifxDestroyBonesManager().

Note:
You cannot instantiate a IFXBonesManager directly. You must use IFXCreateBonesManager() to create an IFXBonesManager_Impl and return it as a IFXBonesManager.
Warning:
Although you can send a new mesh by just using UpdateMesh() the old vertexmap must still be valid for that mesh.
LoadMesh() is an alternative to InitMesh() for stand-alone tests.
