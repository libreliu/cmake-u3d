Detailed Description

This is the main interface for IFXSimulationManager.

Note:
The associated IID is named IID_IFXSimulationManager, and the CID for the component is CID_IFXSimulationManager.

U32  priority,
void *  pUserData,
IFXTaskHandle *  pTaskHandle
)  [pure virtual]
 

This is a convenience method which registers a new simulation task.

Internally this creates a new IFXSimulationInfo and calls IFXTaskManager::AddTask.

Parameters:
pTask The IFXTask which is to be registered.
priority The required priority of the new task.
pUserData A user data pointer.
pTaskHandle Receives a unique task handle which identifies the registration and can be used to unregister the task.
Returns:
Returns an IFXRESULT code. Since this method also calls the IFXTaskManager::AddTask method internally, any of the result codes returned by that method may also be returned by this method.
  • IFX_OK upon success.
  • IFX_E_NOT_INITIALIZED if the component has not been initialized.
  • IFX_E_INVALID_POINTER is returned if pTask or pTaskHandle is NULL.
