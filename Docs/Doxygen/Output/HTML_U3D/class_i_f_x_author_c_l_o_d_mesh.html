Detailed Description

An IFXAuthorMesh derivative that describes a multi-resolution representation of an IFXAuthorMesh.

This representation differs from the IFXAuthorMesh in that its data may be reordered, and it contains additional "update records" which describe how to add or remove mesh vertices, faces, etc. at each mesh resolution change.
