Detailed Description

template<class T>
class IFXAutoPtr< T >

Automatic pointer management class.

This manages a pointer to memory allocated on the heap. After a pointer is assigned to an auto point, it assumes responsibility for deleting it.

This differs from a smart pointer, in that when another auto pointer is given control over the object, the previous owner does not retain any control.
