Detailed Description

(internal) This class shuold be entirely hidden behind IFXList<>.

This implementation of linked list can maintain validity of contexts even when you remove the node it is pointing to. This is done without the lists or nodes retaining information on the contexts that point to it. Every node has an heir that can point to another node in the same list which may or may not have been removed. The heir specifies where to look next for a valid node if this one is invalid. It will follow the chain of heirs until either it reaches a valid node or a NULL pointer, in which case the list is empty. It behaves as though it could update any contexts that were pointing to it, but without the overhead of maintaining pointers to an unlimited number of contexts.
