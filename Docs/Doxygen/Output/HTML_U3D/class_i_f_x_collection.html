virtual IFXRESULT IFXAPI IFXCollection::InitializeCollection (  IFXCollection *  pInCollection  )  [pure virtual]
 

Initializes the Collection.

Precondition:
None; re-initialization is supported.
Postcondition:
If a spatial set is provided, the collection will contain these spatials, if not, it will be empty.
Parameters:
rpInSpatials A reference to the address of the array of spatials to add to the collection.
uInNumberOfSpatials The number of spatials in the rpInSpatial array.
eInType If this parameter is provided, the type of ALL of the provided spatials is assumed to be eInType. If not specified, each spatial is queryied for it's type separately.
Returns:
One of the following IFXRESULT codes:

  • IFX_OK
    No error.

  • IFX_E_OUT_OF_MEMORY
    Not enough memory was available to initialize.

Implemented in CIFXOctreeCollection, CIFXNode, and CIFXSimpleCollection.
