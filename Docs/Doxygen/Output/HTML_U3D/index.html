Introduction

The Universal 3D (U3D) Sample Software is a set of C++ dynamic link run-time libraries to write, read, extend, render and interact with U3D-formatted data. It is intended to demonstrate how to perform such activities and to be used as a starting point for tools and applications to support U3D. Example clients include exporters, converters, importers, authoring packages and interactive viewers.

This software is only sample software and is intended to provide a starting point for ISVs to develop U3D applications (it has received only limited validation and testing). It includes a few usage examples for software developers which are not complete or polished applications. The intention is to allow other companies to innovate and develop solutions appropriate for different target user groups. Refer to the Known Issues section of the Release Notes (located in the Docs directory) for a list of any known deviations from the specification, unimplemented functionality and issues that may be encountered with this software.

Note:
The Universal 3D File Format specification is available from ECMA International* as standard ECMA-363.
