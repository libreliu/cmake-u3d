Detailed Description

This module contains the plug-in specific functions required to be exported from the DL by the IFXCOM component system.

In addition, it also provides common cross-platform startup and shutdown functionality for the DL.

IFXRESULT IFXAPI IFXPluginCanUnloadNow (   ) 
 

This function is used to let component system know if plug-in can be unloaded now or not.

Returns:
If plug-in can be unloaded, it will return the value IFX_OK. Otherwise, it will return the value IFX_E_NOT_DONE.

IFXRESULT IFXAPI IFXPluginRegister (  U32 *  pComponentNumber,
IFXComponentDescriptor **  ppComponentDescriptorList
) 
 

This function provides registration information about plug-in components to the IFXCOM.

Input parameters for this functions should be defined in the caller function.

Parameters:
pComponentNumber Pointer to the number of components in component descriptor list.
ppComponentDescriptorList Pointer to the component descriptor list array (or pointer to the first element pointer of component descriptor list).
Returns:
Upon success, it returns the valuse IFX_OK. If input pointers were not defined, it will return the value IFX_E_PARAMETER_NOT_INITIALIZED.

Variable Documentation

U32 g_countActiveObjects
 

Count of active objects.

This counter is used by IFXPluginCanUnloadNow. If this counter equals 0 it means there is no active objects allocated by plug-in and plug-in can be successfully unloaded.

IFXComponentDescriptor gs_componentDescriptorList[] [static]
 

gs_componentDescriptorList

List of ComponentDescriptor structures for each IFXCOM component exposed by the plug-in module. Look at the IFXPlugin.h" for IFXComponentDescriptor declaration.
