void CIFXDeviceTexUnitOGL::InitData (   )  [protected]
 

Initializes all state settings to default values.

Default values are:

  • RGBInput0: IFX_TEXTURE
  • RGBInput1: IFX_INCOMING
  • RGBInterpolator: IFX_CONSTANT
  • RGB Blend Func: IFX_MODULATE
  • Alpha Input0: IFX_TEXTURE
  • Alpha Input1: IFX_INCOMING
  • Alpha Interpolator: IFX_CONSTANT
  • Alpha Blend Func: IFX_MODULATE
  • Tex Coord Generation: IFX_NONE
  • Texure ID: 1 (Default white texture)
  • Texture Transform: Identity matrix
  • Texture Coordinate Set: -1
