Detailed Description

The IFXWriteBuffer interface is used to write to a file.

Member Function Documentation

virtual IFXRESULT IFXAPI IFXWriteBuffer::Write (  U8 *  pBytes,
U64  position,
U32  count
)  [pure virtual]
 

Write will copy "uCount" bytes from the memory pointed to by "pBytes" to an output buffer starting at "uPosition.".

Parameters:
pBytes Memory location where the bytes to be written come from
position Position within the output buffer to start the write
count Number of bytes of data to write
