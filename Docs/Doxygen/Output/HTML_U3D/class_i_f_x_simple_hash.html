Detailed Description

This is the interface to CIFXSimpleHash.

The simple hash class is a light-weight hash table containing references to IFXUnknown (or IFXUnknown derived) objects. Objects are hashed via a simple U32 key. The internal table size is always a size of the form: (2^n)-1. This allows a very simple hashing function.

IFXUnknown *  pUnk

Add a new IFXUnknown derived object to the hash.

If the specified key uId has already been used, the old data associated with that key is released.

Parameters:

Returns:
An IFXRESULT code.
Return values:
IFX_OK No error.
IFX_E_OUT_OF_MEMORY The object could not be added to the table because there was a memory allocation error.
IFX_E_NOT_INITIALIZED The Initialize() method has not been called successfully yet.
IFX_E_INVALID_POINTER The specified pUnk is NULL.

virtual IFXRESULT IFXAPI IFXSimpleHash::Initialize (  U32  uTableSize  )  [pure virtual]
 

Initialze the hash to the specified table size.

The actual implemented table size will be the closest size to uTableSize that is of the form: (2^n)-1. Calling initialize after data is stored in the table will release all stored data.

Parameters:
uTableSize Input U32 specifying the desired table size.
Returns:
An IFXRESULT error code.
Return values:
IFX_OK No error.
IFX_E_OUT_OF_MEMORY There was an error in allocating space for the table.
