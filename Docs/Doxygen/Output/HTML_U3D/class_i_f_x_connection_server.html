Detailed Description

This interface is used by the connection server to manage connections with its clients in such a fashion that reference count cycles are prevented, system robustness isn't compromised and clients don't have to be aware of dangerous IFXAPI AddRef and Release rule exceptions.

It's supported by the CIFXConnector component. It's expected to be used by connection server components that wish to prevent reference count cycles.

Note:
The associated IID is named IID_IFXConnectionServer.

This interface only manages a single component connection.

virtual IFXRESULT IFXAPI IFXConnectionServer::SetReference (  IFXUnknown *  pUnknown  )  [pure virtual]
 

Method used to establish a connection to the connection server's IFXUnknown interface.

Before the connection server (the component instance that supports the IFXUnknown interface) is destroyed, this method must be called again to disconnect it by specifying a NULL reference.

Parameters:
pUnknown Pointer to the IFXUnknown interface of a component to establish a connection with. Specify a NULL pointer to disconnect a component.
Returns:
Always succeeds and returns IFX_OK.
