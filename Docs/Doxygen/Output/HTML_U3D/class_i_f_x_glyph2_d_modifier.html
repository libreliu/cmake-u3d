Detailed Description

This class provides an interface to construct a 3D extruded model from a set of 2D contours.

It can be used to generate a string of 3D Text. The contour drawing interface matches up with the true type font spec. There are also member functions to control the appearance of the generated models. The depth of the extrusion and beveled faces can be specified.
