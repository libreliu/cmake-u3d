Detailed Description

This is the main interface for IFXBoundHierarchy.

Note:
The associated IID is named IID_IFXBoundHierarchy.

Member Enumeration Documentation

enum IFXBoundHierarchy::SplitType
 

Note:
An enumeration that defines the method used to split the face list.
MIDPOINT - split faces into two buckets at the midpoint of longest side of the box.
MEDIAN - split the face list in half to form two buckets.
SORTED_MEDIAN - sort the face list, then split in half to form two buckets.
