Detailed Description

IFXRender - main render device interface.

This interface represents a single rendering context on one video card. The IFXRenderContext interface represents a virtual window that could span multiple video cards and multiple threads. The implementation for IFXRenderContext uses IFXRender objects to do the actual rendering for each child window that lies completely on one device and runs in one thread.
