Detailed Description

The motion mixer maps tracks from motions to the character.

The simplest method maps all tracks in one motion to bones with matching names.

In addition, you can map specific tracks by name to specific bones, either individually or by sub-hierarchy.

Note that the potential "mixing capabilities" of taking weighted parts of tracks (a non-temporal blend) inside the mixer is not presently supported, so the mixer is really just a mapper.
