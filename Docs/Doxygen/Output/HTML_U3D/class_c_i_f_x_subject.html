IFXRESULT CIFXSubject::Attach (  IFXObserver *  pObs,
U32  interest,
IFXREFIID  rIType = IID_IFXUnknown,
U32  shiftBits = 0
)  [virtual]
 

Associates the provided IFXObserver for callback when the specified aspects of interest change.

Parameters:
pInObserver A pointer to the IFXObserver to become associated.
uInIntrestBits The U32 bit field that describes the aspects of interest whose change will trigger a IFXObserver::Resolve() call.
rIType x.
shiftBits Used for components supporting multiply inherited interfaces each with subject interfaces.
Returns:
One of the following IFXRESULT codes:

  • IFX_OK
    No error.

  • IFX_E_INVALID_POINTER
    pInObserver was NULL.

  • IFX_E_OUT_OF_MEMORY
    Not enough memory is available to resolve the change.
Note:
This method is typically called from the IFXObserver.

void CIFXSubject::PostChanges (  U32  changedBits  )  [virtual]
 

Publishes to attached Observers and ChanageManager that changes have occured.

Parameters:
uInChangedBits The U32 bit field that describes the conceptual change with respect to the published interests.
Note:
Should be called by the IFXSubject on itself when properties change that correlate to the published interests. When PostChanges() is called, all attached Observer Notify() methods are called.
