Detailed Description

This header allows code to use some DX8 types names instead of the DX7 ones when working with just the DX7 SDK headers.

It enables the code to be more similar.

Note:
The defines contained in this header are mostly just an inversion of the defines contained in the Microsoft DirectX 8 SDK header file called dx7todx8.h.
