Detailed Description

template<class T>
class IFXSmartPtr< T >

The IFXSmartPtr object contains a pointer to an IFXUnknown derived object.

The smart pointer will automatically adjust the reference count of the underlying object when necessary (object goes out of scope, assigned to a new object, etc.).
