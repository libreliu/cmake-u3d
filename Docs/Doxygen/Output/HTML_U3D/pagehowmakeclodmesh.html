How To Make A CLOD Mesh (Using AuthorGeomCompiler)

This page describes the process of make a CLOD mesh and how to use the AuthorGeomCompiler component.

About AuthorGeomCompiler

What is it?
  • Utility class used to create an author CLOD resource from an author mesh

Why use it?

  • Makes creating an author CLOD resource easier
  • Encapsulates steps like mesh scrubbing and using CID_IFXAuthorCLODGen

Where is it?

  • In IFXExporting DL and associated project
  • Source code exists physically in Source\RTL\Component\Exporting directory

Component

  • CID - CID_IFXAuthorGeomCompiler
  • Class - CIFXAuthorGeomCompiler

Interface

  • IID - IID_IFXAuthorGeomCompiler
  • Class - IFXAuthorGeomCompiler

Creating an AuthorCLODResource

Here is a summary of the steps that will follow which describe how to create an AuthorCLODResource:
  1. Setup an author mesh
  2. Setup compiler parameters
  3. Compile
  4. Put the author CLOD resource in palette
  5. Optional: Add skeleton, shading modifier, ...

1. Setup an author mesh

  1. Create author mesh component (CID_IFXAuthorMesh) and get its IFXAuthorMesh interface
  2. Call IFXAuthorMesh::Allocate to allocate space for maximum resolution mesh
  3. Populate arrays for faces, positions, normals, texture coordinates, etc.
    • Operate on whole array or individual element

2. Setup compiler parameters

All parameters are either Booleans or values, plus one callback function pointer. They are all contained by the IFXAuthorGeomCompilerParams structure.

  1. Scrub params (IFXAuthorMeshScrubParams)
    • Remove unused or invalid
    • Zero area faces
    • Normalize normals
  2. CLOD params (IFXAuthorCLODGenParam)
    • Merge unconnected
    • Normals mode (none, no updates, parent update, track surface)
  3. Compress params (IFXAuthorCLODCompressionParams)
    • Quality factors
    • Minimum resolution
    • Normals parameters (crease angle, exclude ...)
    • Streaming priority
  4. Progress callback function param (IFXProgressCallback)

3. Compile

  1. Create component CID_IFXAuthorGeomCompiler and get its IFXAuthorGeomCompiler interface
  2. Prepare component by specifying the Scene Graph with the IFXAuthorGeomCompiler::SetSceneGraph function
  3. Call IFXAuthorGeomCompiler::Compile
    1. Input - Name of resource
    2. Input - Author mesh
    3. Output - Author CLOD resource
    4. Input - Encode blocks now flag (can be omitted but it is not recommended)
    5. Input - Compiler parameters IFXAuthorGeomCompilerParams structure (can be omitted but it is not recommended)

4. Put the author CLOD resource in palette

  1. Obtain model GENERATOR palette from Scene Graph with IFXSceneGraph::GetPalette
  2. Use IFXPalette::Add function to add a new palette entry to hold the author CLOD resource
    • Use the exact same name for the resource as during the compile step!
  3. Store resource in palette entry with the IFXPalette::SetResourcePtr function
    • Must use author CLOD resource’s IFXUnknown interface pointer to store it in a palette entry
