virtual void IFXAPI IFXClock::SetAuto (  BOOL  newstate  )  [pure virtual]
 

This method sets the auto-reset flag on the clock.

If auto-reset is true, the clock automatically inverts it's running state upon the next call to IncrementFrame. That is, if it is stopped, it automatically starts, and if it is running, it automatically stops. The auto-reset flag is cleared after it has been triggered, so the clock will not oscillate between running and stopped.

Parameters:
newstate TRUE to set the auto-reset feature, FALSE to clear it.

Implemented in CIFXClock.
