U32 CIFXTaskData::AddRef (  void   )  [virtual]
 

This method increments the reference count for an interface on a component.

Note:
It should be called for every new copy of a pointer to an interface on a given component.
Returns:
It returns a U32 that contains a value from 1 to 2^32 - 1 that defines the new reference count. The return value should only be used for debugging purposes.

IFXRESULT CIFXTaskData::MatchPattern (  IFXTaskData *  pPattern,
BOOL *  pResult
)  [virtual]
 

This method compares this IFXTaskData instance to a reference instance.

The specific matching algorithm and success conditions are implementation defined. If the two instances are determined to match, pResult is set to TRUE; if they do not match, pResult is set to FALSE.

Parameters:
pViewPattern An interface pointer to another IFXTaskData, against which this instance will be compared. If this pointer is NULL, then the match is automatically considered to be a success.
pResult A pointer to a variable which will be set to TRUE or FALSE, according to the results of the match.
Returns:
Returns an IFXRESULT code.
  • IFX_OK upon success.
  • IFX_E_NOT_INITIALIZED if this object has not yet been initialized.
  • IFX_E_INVALID_POINTER if pViewPattern or pResult is NULL.
  • IFX_E_UNSUPPORTED if pViewPattern does not support a required interface.

Implements IFXTaskData.

IFXRESULT CIFXTaskData::QueryInterface (  IFXREFIID  riid,
void **  ppv
)  [virtual]
 

This method provides access to the various interfaces supported by a component.

Upon success, it increments the component's reference count and hands back a pointer to the specified interface.

Note:
A number of rules must be adhered to by all implementations of IFXAPI QueryInterface . For a list of such rules, refer to the IFXCOM usage guidelines.
Returns:
Upon success, IFX_OK is returned. Otherwise, either IFX_E_INVALID_POINTER or IFX_E_UNSUPPORTED are returned.

Implements IFXUnknown.

U32 CIFXTaskData::Release (  void   )  [virtual]
 

This method decrements the reference count for the calling interface on a component.

Note:
If the reference count on a component falls to zero, the component is destroyed.
Returns:
It returns a U32 that contains a value from 1 to 2^32 - 1 that defines the new reference count. The return value should only be used for debugging purposes.

Implements IFXUnknown.
