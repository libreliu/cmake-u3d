Compiles the input IFXAuthorMesh.

This may involve "scrubbing" bad faces and unused mesh data from the mesh, producing a clodolution representation of the mesh, and compressing the mesh into a form optimized for streaming over the Internet. NOTE: Please make sure that you turned srubbing on and set remove zero area faces option on before calling this function. In other case you can experiance wrong behavior with models containing "zero" faces.

Parameters:
rName The name that will be given to the generator created by this operation. This is also the name given the data blocks when they are written to disk.
in_pMesh The input IFXAuthorMesh to compile.
out_ppResource The returned pointer to the resource (IFXAuthorCLODResource) created by this compiler. You must release this resource once you have finished using it.
in_pParams The optional parameters which control the compilation process.
Returns:
An IFXRESULT value.
Return values:
IFX_OK No error.
IFX_E_INVALID_POINTER One (or more) of the input arguments was NULL.
IFX_E_OUT_OF_MEMORY There was not enough memory to create a requested component.
IFX_E_UNDEFINED An error was detected in the compression parameters.
