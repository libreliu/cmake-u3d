IFXINLINE U32 IFXVector4::GetBGRA (   )  const
 

Gets the BGRA representation of the vector as a single integer.

This is useful for setting colors in D3D. This assumes that colors as held in the floating point vector are in the range from 0.0 - 1.0. Colors are clamped to 0.0 - 1.0 range during conversion to U32. The vector is not changed by the clamping.
