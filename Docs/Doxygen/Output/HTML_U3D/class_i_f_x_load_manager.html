Detailed Description

The IFXLoadManager interface is used to load a U3D File from an IFXReadBuffer into the scenegraph.

virtual IFXRESULT IFXAPI IFXLoadManager::Cancel (   )  [pure virtual]
 

The Load Manager will stop any load or reload operation in progress.

If the Load Manager is registered with the scheduler, the Load Manager will unregister with the scheduler. The Load Manager will clear internal state. If the Load Manager was initialized, the Load Manager will remain initialized.

virtual IFXRESULT IFXAPI IFXLoadManager::Load (  IFXReadBuffer *  pReadBuffer,
const BOOL  isAtomicLoad,
const IFXString &  rScope,
const IFXCollisionPolicy  collisionPolicy,
const U32  loadId,
const IFXObjectFilters &  rObjectFilters,
const IFXString &  rWorldAlias,
const BOOL  isExternalLoad
)  [pure virtual]
 

The Load Manager will load the information from pReadBuffer into objects in the scene graph obtained from the core services and create those objects as needed.

Parameters:
pReadBuffer The Load Manager will start loading information from pReadBuffer.
bAtomicLoad If bAtomicLoad is true, the Load Manager will load the entire pReadBuffer is possible. If bAtomicLoad is false, the Load Manager will attempt to limit the amout of time spent in the Load call; the Load call may return without loading the entire pReadBuffer. If needed, the Load Manager will register with the scheduler to get additional time to complete the loading of the pReadBuffer.
sScope Current name scope used for names collision resolution.
uCollisionPolicy Collision resolution policy to use when different objects has same names.
uLoadId Identifier of current LoadManager. Serves for names collision resolution.
rObjectFilters Set of filters to apply when objects are loaded.
rWorldAlias Default node alias in external file.
bExternalLoad Indicates that current file is an external referenced files.

Implemented in CIFXLoadManager.
