Detailed Description

The IFXRenderCaps object contains a set of flags and values that determine the capabilities of a rendering device.

These can be used to make determinations on optimal rendering algorithms. All data members beginning with m_b are one bit values. The 1 (TRUE) value specifies that the capability is present, the 0 (FALSE) value means that the capability is not present.
