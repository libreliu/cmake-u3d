Detailed Description

The implementation of this class handles the loading and unloading of the DirectX7 DDraw library, and querying for a function pointers to DirectDrawEnumerateEx and DirectDrawCreateEx.

These are the only entry points in ddraw.dll that needs to be used to fully use DX7 D3D.
