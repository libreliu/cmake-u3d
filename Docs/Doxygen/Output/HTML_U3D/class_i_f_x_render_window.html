IFXRESULT IFXRenderWindow::SetDTS (  BOOL  bDTS  ) 
 

Specifies whether or not to use direct to screen (DTS) mode or not.

If bDTS is TRUE, rendering occurs normally (i.e. back buffer is copied directly to the screen on IFXRenderContext::Swap()). If bDTS is FALSE, then the data is copied from the back buffer to the memory location specified by SetWindowPtr().

Parameters:
bDTS Input boolean specifying whether or not to use direct to screen rendering.
Returns:
An IFXRESULT code
Return values:
IFX_OK No error.

IFXRESULT IFXRenderWindow::SetWindowSize (  const IFXRect &  rcWindow  ) 
 

Sets the size (and position) of the renderable area.

If full screen mode is being used, then the iWidth and iHeight parameters specify the screen resolution. If windowed mode is used, then all four parameters define where (relative to the passed in window or memory location pointer) the rendering occurs. This, in effect, creates a sub window region for the renderer to use. The upper left corner of the "parent" area is (0,0).

Parameters:
rcWindow Input IFXRect specifying the size and position of the window relative to the parent window. If fullscreen mode is enabled, then only the m_Width and m_Height of rcWindow are used.
Returns:
An IFXRESULT code
Return values:
IFX_OK No error.
IFX_E_INVALID_RANGE If full screen mode is used, then the rcWindow.m_Width and/or rcWindow.m_Height parameters are not valid display resolutions. If windowed mode is used, then the rcWindowm_Width or rcWindow.m_Height parameters are <= 0.
