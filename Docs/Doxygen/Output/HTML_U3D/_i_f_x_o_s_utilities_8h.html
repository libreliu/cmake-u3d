Detailed Description

This header defines an abstraction layer for general OS specific utilities.

The implementation of these utilities resides in a source file named after the OS. To easily identify functions belonging to this module, all function names have been prefixed with "IFXOS_".

Note:
To port to another operating system, rewrite the all of the IFXOS*.h services. Do not put any OS specific includes or types in this header!

void IFXAPI_CALLTYPE IFXOSSleep (  U32  milliseconds  ) 
 

This function is used to put the current thread to sleep for the specified number of milliseconds (at a minimum).

Note:
A value of zero causes the thread to relinquish the remainder of its time slice to any other thread of equal priority that is ready to run. If there are no other threads of equal priority ready to run, the function returns immediately, and the thread continues execution.
