Detailed Description

The IFXEncoderX interface is implemented by various encoders used by the CIFXWriteManager.

Each instance of an encoder is used to create data blocks for one instance of a scene graph object. Each type of encoder is used to create data blocks for a particular type of object. The data blocks contain encoded state information for the object.
