Detailed Description

The IFXRenderServices object is a singleton rendering manager.

It keeps track of registered renderer types and allocated render device objects. It also holds default values for various rendering states (i.e. internal texture pixel format). The IFXRenderServices object must be created before any IFXRenderContext objects, and must be destroyed after all IFXRenderContext objects have been destroyed.

virtual IFXRESULT IFXAPI IFXRenderServices::RegisterRenderer (  U32  uRenderId,
IFXREFCID  cidRender,
IFXREFCID  cidRenderDevice
)  [pure virtual]
 

Registers a renderer type with the system.

Each renderer type (DirectX8, OpenGL, Software, etc.) is described by a renderer ID, along with class ID's (CID's) for the type specific implementation of IFXRender and IFXRenderDevice interfaces (these implementations should be derived from CIFXRender and CIFXRenderDevice respectively).

Parameters:
uRenderId Input render ID that will be associated with this renderer type. This value must not be 0.
cidRender Input IFXREFCID (class ID) that identifies the renderer specific implementation of IFXRender. This class should be derived from CIFXRender.
cidRenderDevice Input IFXREFCID (class ID) that identifies the renderer specific implementation of IFXRenderDevice. This class should be derived from CIFXRenderDevice.
Returns:
An IFXRESULT value.
Return values:
IFX_OK No error.
IFX_E_INVALID_RANGE Either the value of uRenderId is 0, or the given uRenderId has already been registered.

Implemented in CIFXRenderServices.

virtual IFXRESULT IFXAPI IFXRenderServices::SetPixelAspectRatio (  F32  fRatio  )  [pure virtual]
 

Sets the pixel aspect of the device.

The pixel aspect can be changed when the display device used has a pixel aspect other than 1.0. For instance, you are using a standard 4:3 resolution (1600x1200) on a widescreen 16:9 monitor. In that case, you would use a pixel aspect ratio of 1.333333. The initial value is 1.0.

Parameters:

Returns:
An IFXRESULT value.
Return values:

IFX_E_INVALID_RANGE The value specified with fRatio was either 0.0 or negative.
