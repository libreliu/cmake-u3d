Detailed Description

A structure whose parameters control the mesh scrubbing operation.

By default, bInvalidIndices, bUnusedAttributes, and RemoveUnusedMaterials are true if no other conditions are specified.

BOOL IFXAuthorMeshScrubParams::bInvalidIndices
 

Checks for out of range indices on any of the Face attribute arrays, Faces with invalid Indices will be removed.

Additionally this step will set unused face attributes to IFX_AUTHOR_INVALID_INDEX;

BOOL IFXAuthorMeshScrubParams::bNormalizeNormals
 

Makes sure all normals are unit vectors.

BOOL IFXAuthorMeshScrubParams::bOperatateInPlace
 

Don't copy the mesh before scrubbing it.

This will cause this input data to be overwritten.
