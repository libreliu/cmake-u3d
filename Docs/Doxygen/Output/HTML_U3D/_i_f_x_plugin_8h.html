Detailed Description

This header defines the functions that need to be exported from each IFXCOM plug-in DLL.

It also defines the type of the common factory function that is implemented for each supported IFXCOM component.

Extra method to support registration of IFXCOM components need to be exposed by plug-in DLLs.
