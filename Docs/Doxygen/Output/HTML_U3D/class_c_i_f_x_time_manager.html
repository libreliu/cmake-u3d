IFXRESULT CIFXTimeManager::RegisterTask (  IFXTask *  pTask,
U32  start,
U32  period,
U32  duration,
IFXTaskHandle *  pTaskHandle
)  [virtual]
 

Creates a new notification that is controlled by a timer.

This is similar to the SetTimer method, except that the notification is auto-generated rather than being specified by the client. Instead of returning the TimerId, this method returns the taskhandle of the notification handler.

Parameters:
pTask The IFXTask which is to be handle events generated by this timer.
start starting time for this timer in milliseconds, relative to the present.
period period for this timer in milliseconds.
duration duration for this timer in milliseconds. A duration of zero means infinity.
pTaskHandle Receives the task handle that uniquely identifies this timer/notification pair.
Returns:
Returns an IFXRESULT code..
  • IFX_OK upon success.
  • IFX_E_NOT_INITIALIZED if the Time Manager has not been initialized.
  • IFX_E_INVALID_POINTER is returned if pTask or pTaskHandle is invalid.
  • IFX_E_OUT_OF_MEMORY is returned if the new timer cannot be allocated.
Note:
Timers created by this method should be deleted using the UnregisterTask method. Unregister task properly kills the timer and deletes the associated notification.

Implements IFXTimeManager.

This method creates a new timer.

This does not create an associated Notification for the timer; the notification to be generated must be specified by the id parameter.

Parameters:
id The IFXNotificationId for this timer. This id will be supplied to the NotificationManager when this timer is triggered, and may be caught by notification handlers.
start starting time for this timer in milliseconds, relative to the present.
period period for this timer in milliseconds.
duration duration for this timer in milliseconds. A duration of zero means infinity.
pTimerId Receives a unique id for this timer. This id can be used to kill the timer prematurely.
Returns:
Returns an IFXRESULT code..
  • IFX_OK upon success.
  • IFX_E_NOT_INITIALIZED if the Time Manager has not been initialized.
  • IFX_E_INVALID_POINTER is returned if pTimerId is invalid.
  • IFX_E_OUT_OF_MEMORY is returned if the new timer cannot be allocated.
  • IFX_E_OUT_OF_RANGE indicates that id is an invalid IFXNotificationId.
