Detailed Description

The IFXMeshGroup class groups multiple IFXMeshes.

Typically a MeshGroup represents a single object that was partitioned into multiple IFXMeshes.

The MeshGroup stores only pointers to meshes, so a single mesh may be shared by many different MeshGroups.
