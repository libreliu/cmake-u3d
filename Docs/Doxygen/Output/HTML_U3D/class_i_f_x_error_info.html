Detailed Description

This is the main interface for IFXErrorInfo.

When an IFXTask::Execute method produces an error, an error notification is generated by submitting information about the error, represented by a component that supports this interface. The error notification can be handled by an IFXTask handler registered with the Notification Manager; the handler should be registered as IFXNotificationType_Error and the IFXNotificationId should be equal to the IFXTaskHandle of the registered task that will be monitored. To monitor all registered tasks, use IFXNotificationId_Nil. The handler will receive a reference to an IFXNotificationInfo interface (supported by a simple data holding component) as the argument to its Execute method. If no IFXTask is registered to handle errors, the error is discarded.

Note:
The associated IID is named IID_IFXErrorInfo, and the CID for this component is CID_IFXErrorInfo.
