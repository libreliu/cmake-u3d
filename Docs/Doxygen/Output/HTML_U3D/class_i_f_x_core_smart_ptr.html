Detailed Description

Abstract base class for IFXSmartPtr.

The implementation is split between the two classes to avoid as much code generation as possible with new template specializations of IFXSmartPtr<T>.

IFXRESULT IFXAPI IFXCoreSmartPtr::QI (  IFXSmartPtr< S > &  rPtr,
IFXREFIID  iid
)  const [inline]
 

This will attempt to query interface the underlying object for an interface of type S, and fill the rPtr argument with that new interface pointer.

If the method succeeds, the total reference count of the underlying object will be incremented (since rPtr now owns a reference).

Parameters:
@rPtr IFXSmartPtr of a different interface 'S' than this instance. If the interface type S can by queried by the underlying object, rPtr will be filled with the pointer to the queried interface.
Returns:
An IFXRESULT value.
Return values:
IFX_OK No error.
IFX_E_INVALID_RANGE The interface type of rPtr is not compitable with this object.

Recast (QueryInterface) the underlying object to an interface with IID guid, and place the results in rvPtr.
