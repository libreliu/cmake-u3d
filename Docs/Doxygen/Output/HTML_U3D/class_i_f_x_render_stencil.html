Detailed Description

This interface encapsulates all settings relating to the stencil buffer.

It is used to pass stencil settings into an IFXRender object using IFXRenderContext::SetStencil(). Usage of the stencil buffer can be enabled with IFXRenderContext::Enable(IFX_STENCIL). For this to work, you must have a stencil buffer allocated.

See also:
IFXRenderContext::SetStencil(), IFXRenderWindow

IFXINLINE IFXenum IFXRenderStencil::GetFailureOp (   )  const
 

Gets the stencil test fail operation.

Returns:
The stencil test fail operation.
Return values:
IFX_KEEP Do nothing to the stencil buffer.
IFX_REPLACE Replace the stored stencil value with the reference value.
IFX_INCR Increment the stencil buffer. This will wrap to 0 on overflow.
IFX_DECR Decrement the stencil buffer. This will wrap to Max (255) on overflow.
IFX_INCRSAT Increment the stencil buffer. This will not not wrap.
IFX_DECRSAT Decrement the stencil buffer. This will not wrap.
IFX_INVERT Do a binary inversion (1->0, 0->1) on the stencil buffer.

IFXINLINE void IFXRenderStencil::InitData (   ) 
 

Initialize all data to default values.

Default values are:

  • Failure Op: IFX_KEEP
  • ZFailure Op: IFX_KEEP
  • Pass Op: IFX_KEEP
  • Stencil Func: IFX_ALWAYS
  • Stencil Reference Value: 0
  • Stencil Test Mask: 0xff
  • Stencil Write Mask: 0xff

IFXINLINE IFXRESULT IFXRenderStencil::SetFailureOp (  IFXenum  eOp  ) 
 

Sets the operation to take place when the stencil test fails.

Some video cards and APIs don't support IFX_INCR or IFX_DECR. In this case, they will use IFX_INCRSAT and IFX_DECRSAT.

Parameters:
eOp Input IFXenum specifying the stencil test fail operation. Valid values are:
  • IFX_KEEP: Do nothing to the stencil buffer.
  • IFX_REPLACE: Replace the stored stencil value with the reference value.
  • IFX_INCR: Increment the stencil buffer. This will wrap to 0 on overflow.
  • IFX_DECR: Decrement the stencil buffer. This will wrap to Max (255) on overflow.
  • IFX_INCRSAT: Increment the stencil buffer. This will not not wrap.
  • IFX_DECRSAT: Decrement the stencil buffer. This will not wrap.
  • IFX_INVERT: Do a binary inversion (1->0, 0->1) on the stencil buffer.
Returns:
An IFXRESULT error code
Return values:
IFX_OK No error.
IFX_E_INVALID_RANGE The specified operation is invalid.

IFXINLINE IFXRESULT IFXRenderStencil::SetStencilFunc (  IFXenum  eFunc  ) 
 

Sets the comparison function used for the stencil test.

The comparison takes place between the current value of the stencil buffer and the reference value.

Parameters:
eFunc Input IFXenum specifying the stencil test comparison function. Valid values are:
  • IFX_NEVER: The test never passes.
  • IFX_LESS: Passes when the reference value is less than the stencil value.
  • IFX_GREATER: Passes when the reference value is greater than the stencil value.
  • IFX_EQUAL: Passes when the reference value is equal to the stencil value.
  • IFX_NOT_EQUAL: Passes when the reference value is not equal to the stencil value.
  • IFX_LEQUAL: Passes when the reference value is less than or equal to the stencil value.
  • IFX_GEQUAL: Passes when the reference value is greater than or equal to the stencil value.
  • IFX_ALWAYS: The test always passes.

An IFXRESULT error code

IFXINLINE IFXRESULT IFXRenderStencil::SetTestMask (  U32  uMask  )

IFXINLINE IFXRESULT IFXRenderStencil::SetWriteMask (  U32  uMask  ) 
 

Sets the write mask for the stencil buffer.

This mask is a binary bit by bit write enable mask for the stencil buffer. Only those stencil bits whose mask bit is 1 will be written to. A uMask of 0x0 will disable all stencil updates.

Parameters:
uMask Input U32 specifying the stencil buffer write mask. This value is clamped to the maximum stencil value (probably 0xff, or 255).
Returns:
An IFXRESULT error code
Return values:
IFX_OK No error.
