Detailed Description

The implementation of this class handles the loading and unloading of the DirectX8 D3D library, and querying for a function pointer to Direct3DCreate8.

This is the only entry point in d3d8.dll that needs to be used to use DX8 D3D.

We also use a DirectDraw7 object for implementing VSync. This is much easier (and seems to work better) than using DX8's VSync routines. Therefore, this object also loads the DDraw.dll and retrieves a function pointer to DirectDrawCreateEx.
