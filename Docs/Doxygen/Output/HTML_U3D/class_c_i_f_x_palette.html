IFXRESULT CIFXPalette::AddHidden (  IFXString *  pPaletteEntryName,
U32 *  pPaletteEntryId
)  [virtual]
 

This function adds a new entry to the palette marked as hidden.

Hidden entries are not exposed to the author and not written to the file. If an entry with the given name already exists, a unique name is created.

Parameters:
pPaletteEntryName Pointer to the name for the palette entry.
pPaletteEntryId A U32 pointer indicating the index where the entry was added.
Returns:
IFXRESULT
  • IFX_OK - No error.
  • IFX_E_INVALID_POINTER - Invalid pointer passed in.
  • IFX_E_NOT_INITIALIZED - Internal palette has not been initialized.
  • IFX_E_CANNOT_CHANGE - Trying to add an entry at reserved index zero.
  • IFX_E_OUT_OF_MEMORY - Failed memory allocation.

Note:
If it was necessary to create a unique name for the new hidden palette entry, keep track of the entry id handed back in case it is needed later. Find function will not be able to locate the entry using the same name specified to this function because it is not aware of the unique entry name that was actually used.
