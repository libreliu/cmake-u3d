IFXINLINE void CIFXDeviceTexUnit::InitData (   ) 
 

Initializes all state settings to default values.

Default values are:

  • RGBInput0: IFX_TEXTURE
  • RGBInput1: IFX_INCOMING
  • RGBInterpolator: IFX_CONSTANT
  • RGB Blend Func: IFX_MODULATE
  • Alpha Input0: IFX_TEXTURE
  • Alpha Input1: IFX_INCOMING
  • Alpha Interpolator: IFX_CONSTANT
  • Alpha Blend Func: IFX_MODULATE
  • Tex Coord Generation: IFX_NONE
  • Texure ID: 1 (Default white texture)
  • Texture Transform: Identity matrix
  • Texture Coordinate Set: -1

IFXRESULT CIFXDeviceTexUnit::SetTextureId (  U32  uTexId  ) 
 

Sets the texture to use for this texture unit.

Since the texture validity is not known until render time, this method does not check to see if the texture ID is valid.

Parameters:
uTexId Input U32 specifying the registered texture ID for the texture to use. Specifying 0 will not associate a texture with this texture unit.
Returns:
An IFXRESULT error code.
Return values:
IFX_OK No error.
