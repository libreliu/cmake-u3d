Detailed Description

The mesh compiler converts AuthorMesh into IFXMesh.

This compiler handles static or streaming CLOD meshes. The compiler also handles meshes that a have a static low resolution portion which is followed streaming high resolution portion. This is controlled by settings on the AuthorMesh which must be made before calling the compiler.

This is typically called many times to incrementally convert the mesh to IFX format.

This should only be called after calling InitCompile. Converts available AuthorMesh data to IFXMesh. The available amount is indicated by calling SetMaxResolution on the AuthorMesh that was passed in during InitStreamCompile.
