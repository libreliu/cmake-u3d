Detailed Description

Interface to the CIFXIDManager component.

This class distributes and recycles ID numbers for any purpose.

virtual IFXRESULT IFXAPI IFXIDManager::GetId (  U32 &  uOutId  )  [pure virtual]
 

Retrieves a new ID number.

This will register the number returned in uOutId as being used. It will not be returned from GetId() again until it has been released with ReleaseId(). GetId() will never return the ID number 0.

Parameters:
uOutId Output U32 to receive the new ID number.
Returns:
An IFXRESULT code.
Return values:
IFX_OK No error.
