Performance Optimizations

The U3D Sample Software has a variety of performance optimizations.

While the architecture provides a crucial high-level form of optimization, there are also lower-level optimizations included for IA-32 based platforms. In addition, if the Intel® Integrated Performance Primitives (IPP) DLs (http://www.intel.com/software/products/ipp/index.htm) are installed and present in the U3D DLs PATH there will be an additional automatic run-time performance improvement for matrix functionality, texture image reformatting & resizing, and basic memory management.
