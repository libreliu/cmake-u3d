Detailed Description

This is the implementation of a component that can be registered as a task which will call a client defined callback function whenever the task is executed.

It supports the following interfaces: IFXTaskCallback, IFXTask and IFXUnknown.

Note:
This component's id is named CID_IFXTaskCallback.

This component can be instantiated multiple times.
