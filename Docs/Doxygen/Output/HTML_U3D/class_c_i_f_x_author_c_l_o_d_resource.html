void CIFXAuthorCLODResource::GetNormalCreaseParameter (  F32 &  rNormalCreaseParameter  )  [virtual]
 

Returns the normal crease parameter, which is used during compression.

Normals at the same position which are closer than the crease parameter are merged. The crease parameter is the dot product between the normals (-1 to 1).

Parameters:
rNormalCreaseParameter The returned reference to the normal crease parameter.

Implements IFXAuthorCLODResource.

void CIFXAuthorCLODResource::GetNormalTolerance (  F32 &  rNormalTolerance  )  [virtual]
 

Returns the normal tolerance, which is used during compression.

Normals smaller than this value will be considered the same. Larger values result in a smaller render mesh but at the sacrifice of accuracy. Smaller values are accurate but result in a bulky mesh. The tolerance is expressed as the dot product between the two normals (-1 to 1).

Parameters:
rNormalTolerance The returned reference to the normal tolerance.

Implements IFXAuthorCLODResource.

void CIFXAuthorCLODResource::GetNormalUpdateParameter (  F32 &  rNormalUpdateParameter  )  [virtual]
 

Returns the normal update parameter, which is used during compression.

Corrections to the predicted normal (determined during the compression process) that are smaller than the update value are dropped. The update parameter is expressed as a dot product between the original authored normal and the predicted normal (-1 to 1).

Parameters:
rNormalUpdateParameter The returned reference to the normal update parameter.
