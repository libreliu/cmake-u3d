Detailed Description

This class is used as a corner-based iterator for walking the surface of a model.

The iterator stores the current mesh index, face index, and corner index. It provides methods for going to the clockwise or counterclockwise corner on a face and for jumping across to the adjacent face. (nonlinear, corner-based, iterator)
