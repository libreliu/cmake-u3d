Detailed Description

This is the main interface for IFXBoundVolume - the interface from which all bounding volume objects used by the bounding hierarchy are derived.

Member Function Documentation

virtual IFXRESULT IFXAPI IFXBoundVolume::IntersectBound (  IFXBoundVolume *  pBound  )  [pure virtual]
 

This method takes as input another bounding volume and tests it and the current one for overlap.

In the current implementation of the bounding hierarchy the bounding volumes are oriented bounding boxes.

Parameters:
pBound A pointer an IFXBoundVolume.
Returns:
Returns an IFXRESULT code.
  • IFX_OK upon success.

  • IFX_E_INVALID_POINTER is returned if an invalid pointer is passed for pBoundVolume.

Implemented in CIFXAxisAlignedBBox.

This method takes as input another bounding volume and tests it and the current one for overlap using box/sphere overlap tests.

The pBound passed in always refers to the sphere object.
