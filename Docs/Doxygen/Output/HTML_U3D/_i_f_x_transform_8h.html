Detailed Description

3D Transform with JIT auto-conversions.

Wraps most IFXMatrix4x4 and IFXQuaternion functions to do automatic JIT conversions and control validity flags. The lack of virtuals is intentional for speed.

Uses modeling-optimized 3x4 versions of IFXMatrix4x4 operations.

To get data from the OpenGL current matrix: glGetFloatv(GL_MODELVIEW_MATRIX,transform.GetMatrixData(WRITEONLY));

To put data into the OpenGL current matrix: glLoadMatrixf(transform.GetMatrixData(READONLY));

To use relative to the OpenGL current matrix: glMultMatrixf(transform.GetMatrixData(READONLY));

The last two require being in GL_MODELVIEW mode like with: glMatrixMode(GL_MODELVIEW);

Scaling is a recent feature with minimal testing.
