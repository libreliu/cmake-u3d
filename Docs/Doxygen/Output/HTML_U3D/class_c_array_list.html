Detailed Description

template<class T>
class CArrayList< T >

CArrayList is a simple data structure that manages a growing array and has a somewhat STL-like list interface.

This data structure provides cleaner code, by encapsulating code complexity, and increases speed.

Note:
Removal functionality does not maintain element ordering.

No arbitrary insertion functionality is provided.
