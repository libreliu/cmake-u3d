Detailed Description

The IFXResManager class is used to change the resolution of a single IFXMesh.

If an instance of this class is being used by an IFXCLODManager then you should not change it's resolution here since you may throw off the synchronization provided by the CLODManager.

Use this class to change the resolution of a mesh with only one material.

Warning:
Initialize must be called before any other method is called.

the input mesh is assumed to be at full resolution.
