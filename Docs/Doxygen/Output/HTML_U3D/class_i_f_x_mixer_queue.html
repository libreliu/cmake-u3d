virtual IFXRESULT IFXAPI IFXMixerQueue::Push (  const IFXString &  name,
F32 *  localOffset,

F32 *  localEndTime,
F32 *  timeScale,
BOOL *  loop,
BOOL *  sync,
BOOL  delaymap = false,
BOOL  tail = false
)  [pure virtual]
 

Pushes a new motion onto the head of the stack.

If a motion is found by the given name, is is activate with the given offset, cropped time bounds, scaling, loop flag, and sync flag. If delaymap is true, the system will wait until the latest moment possible to actually do the motion lookup in case it is not available yet. If tail is true, the motion is placed at the tail of the queue.
