Detailed Description

This is the interface encapsulating an object used to pass values relating to fog into an IFXRender object.

All fog related states are contained in this interface. To enable fog, use IFXRenderContext::Enable(IFX_FOG).

See also:
IFXRenderContext::SetFogParams(), IFXRenderContext::Enable(), IFXRenderContext::Disable()

IFXINLINE IFXRESULT IFXRenderFog::SetFar (  F32  fFar  ) 
 

Sets the distance at which fog density reaches maximum.

Note, density for fog when using the IFX_FOG_EXP or IFX_FOG_EXP2 modes will never reach full 1.0. This distance specifies where the fog density hits 0.99 in these modes. When the fog mode is IFX_FOG_LINEAR, this distance does specify the full 1.0 fog density distance.

Parameters:
fFar Input F32 specifying the fog saturation distance. This value is clamped to a minumum of 0.
Returns:
An IFXRESULT error code
Return values:
IFX_OK No error.

IFXINLINE IFXRESULT IFXRenderFog::SetMode (  IFXenum  eMode  ) 
 

Sets the method for calculating fog density.

Parameters:
eMode Input IFXenum specifying the method for how fog is calculated. Valid options are:
  • IFX_FOG_LINEAR: Fog density has linear falloff from the LinearNear distance to the Far distance.
  • IFX_FOG_EXP: Fog density has exponential falloff from the camera to the Far distance.
  • IFX_FOG_EXP2: Fog density has exponential squared falloff from the camera to the Far distance.

An IFXRESULT error code
Return values:
