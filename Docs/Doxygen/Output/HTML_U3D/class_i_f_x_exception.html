Detailed Description

This class is the one type of exception explicitly thrown by IFX code.

It supports both IFXRESULT codes and string messages. Helpful utilities commonly used with this class are contained in the file IFXCheckX.h.

Example catch usage:
			try
			{
				SomethingX();
			}
			catch( IFXException& rException )
			{
				rException.Log();
				throw;
			}
Example throw usage:
			SomethingX()
			{
				// When only a string is specified, the associated 
				// IFXRESULT code will be IFX_E_UNDEFINED.
				throw IFXException( L"Something broke!" );
			}
or 
			SomethingX()
			{
				throw IFXException(
						IFX_E_OUT_OF_MEMORY,
						L"System memory request failed." );
			}
