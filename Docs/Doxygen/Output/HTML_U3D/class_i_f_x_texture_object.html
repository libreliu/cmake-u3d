Texture map type can be one of two types,IFX_TEXTURE_2D or IFX_TEXTURE_CUBE. If texture created for use as cubic environment mapping, it needs to set as IFX_TEXTURE_CUBE. When creates the Texture object, the default texture type sets to IFX_TEXTURE_2D.

Returns:
An IFXRESULT code.
Return values:
IFX_OK Successful

Implemented in CIFXTextureObject.

virtual IFXRESULT IFXAPI IFXTextureObject::SetWidth (  U32  uValue  )  [pure virtual]
 

This method sets the new width for the texture object.

If the new value is not power of 2, then it will set to power of 2 and resize the texture image.

Parameters:
U32 uValue Sets uValue as the width for the texture object. If the uValue is not power of 2, it will convert to power of 2 in next UpdateAndMakeCurrent().
Returns:
An IFXRESULT code.
Return values:
IFX_OK Successful.
IFX_E_INVALID_RANGE One or more parameters are out of range.

Implemented in CIFXTextureObject.
