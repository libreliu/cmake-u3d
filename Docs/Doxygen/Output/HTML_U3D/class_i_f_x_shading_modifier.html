virtual IFXRESULT IFXAPI IFXShadingModifier::SetElementShaderList (  U32  uInElement,
IFXShaderList *  in_pShaderList,
BOOL  isValid = TRUE
)  [pure virtual]
 

This method sets shader set for particular renderable lement.

Parameters:
uInElement Index of element in renderable group.
in_ppShaderList A pointer to the shader set.
isValid If TRUE and in_pShaderList is NULL then it means that shading modifier is a pass-through modifier for this renderable element. If FALSE and in_pShaderList is NULL then shading data will be cut at uInElement. If TRUE and in_pShaderList is not NULL then shader list is applied to the selected renderable element. If FALSE and in_pShaderList is not NULL then shader list is applied to all renderable elements from 0 to uInElement.
Returns:
IFXRESULT
Return values:
IFX_OK - No Error.
IFX_E_INVALID_RANGE The uInElement index is larger than the renderable element count

Implemented in CIFXShadingModifier.
