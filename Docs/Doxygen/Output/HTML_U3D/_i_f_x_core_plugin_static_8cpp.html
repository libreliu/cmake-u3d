Detailed Description

This module overrides the default C++ memory operators.

It is part of IFXCorePluginStatic library which defines common functionality for all extension plug-ins.

Function Documentation

void operator delete (  void *  pMemory  ) 
 

Overloaded global memory operator.

Note:
Every plug-in DL should be statically linked to the IFXCore dynamic library and can use IFXAllocate, IFXDeallocate and IFXReallocate from it. Memory operators are overloaded here for the plug-in DL by using IFXAllocate and IFXDeallocate. Thus plug-in DL uses the same memory functions as IFXCore uses.

Variable Documentation

U32 g_countActiveObjects = 0
 

Count of active objects.

This counter is used by IFXPluginCanUnloadNow. If this counter equals 0 it means there is no active objects allocated by plug-in and plug-in can be successfully unloaded.
