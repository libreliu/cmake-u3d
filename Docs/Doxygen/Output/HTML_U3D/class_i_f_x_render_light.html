Detailed Description

Param struct for storing various light parameters.

This is used to pass light parameters into an IFXRender object.

IFXINLINE void IFXRenderLight::InitData (   ) 
 

Initializes all data members to default values.

Default values for IFXRenderLight are:

  • Type: IFX_DIRECTIONAL
  • Ambient: IFXVector4(0,0,0,0)
  • Diffuse: IFXVector4(1,1,1,1)
  • Specular: IFXVector4(1,1,1,1)
  • Intensity: 1.0
  • Attenuation: IFXVector3(1,1,1)
  • World Matrix: Identity matrix
  • Spot Decay: TRUE
  • Spot Inner Angle: 0.0
  • Spot Outer Angle: 45.0
  • Light Range: sqrt(FLT_MAX)

Reimplemented in CIFXDeviceLight, CIFXDeviceLightDX7, CIFXDeviceLightDX8, CIFXDeviceLightNULL, and CIFXDeviceLightOGL.

IFXINLINE IFXRESULT IFXRenderLight::SetAttenuation (  const IFXVector3 &  vAtten  ) 
 

Sets the constant, linear, and quadratic attenuation factors of the light.

Attenuation from the light to the vertex is defined by the formula: 1 / ( C + Ld + Qd^2 ), where:

  • C - constant attenuation
  • L - linear attenuation
  • Q - quadratic attenuation
  • d - distance from vertex to light

You must not specify (0,0,0).

Parameters:
vAtten Input attenuation factor: vector(C,L,Q).
Returns:
An IFXRESULT error code.
Return values:
IFX_OK No error.
IFX_E_INVALID_RANGE The specified attenuation is invalid (all 0's).

IFXINLINE IFXRESULT IFXRenderLight::SetIntensity (  F32  fIntensity  ) 
 

Sets the light intensity factor.

This is a scalar floating point value this is multiplied by the ambient, diffuse, and specular colors before they are sent to the rendering system.

This value can be used to approximate super-saturating lights and negative lights.

IFXINLINE IFXRESULT IFXRenderLight::SetRange (  F32  fRange  ) 
 

Sets the range of influence of the light.

The light range is valid for IFX_SPOT and IFX_POINT lights. The maximum value is sqrt(FLT_MAX). If a light is used to light geometry further away than fRange, the light provides no contribution to the lighting process.

Parameters:
fRange Floating point value specifying world space units for the light range. Must a positive value less than sqrt(FLT_MAX).
Returns:
An IFXRESULT error code.
Return values:
IFX_OK No error.
IFX_E_INVALID_RANGE The specified @ fRange is invalid.
