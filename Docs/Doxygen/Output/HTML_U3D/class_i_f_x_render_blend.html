Detailed Description

This is the interface to an object that provides frame buffer blend and alpha test parameters to an IFXRenderContext interface.

This encapsulates all state settings for alpha blending and alpha test.

IFXINLINE IFXRESULT IFXRenderBlend::SetBlendFunc (  IFXenum  eFunc  ) 
 

Sets the blend function between rendered pixels and the existing framebuffer.

Alpha blending can be enabled by calling: IFXRenderContext::Enable(IFX_FB_BLEND).

Parameters:
eFunc Input IFXenum specifying the desired blending function. Valid values are:
  • IFX_FB_ADD: Add the RGB components into the framebuffer
  • IFX_FB_MULTIPLY: Multiply the RGB components into the framebuffer
  • IFX_FB_ALPHA_BLEND: Linear blend the RGB components into the framebuffer based on the rendered alpha value.
  • IFX_FB_INV_ALPHA_BLEND: Linear blend the RGB components into framebuffer based on the inverse (1.0 - a) of the rendered alpha.
  • IFX_FB_ADD_BLEND: Multiply the alpha values into the source color values, then add to the frame buffer.
  • IFX_FB_INVISIBLE: Use the existing framebuffer values. This is useful to render into the Depth buffer only.
Returns:
An IFXRESULT error code.
Return values:
IFX_OK No error.
IFX_E_INVALID_RANGE The specified comparison function is invalid.
See also:
IFXRenderContext::Enable(), IFXRenderContext::Disable()

IFXINLINE IFXRESULT IFXRenderBlend::SetTestFunc (  IFXenum  eFunc  ) 
 

Sets the comparison function used for alpha test.

Alpha test can be enabled by calling: IFXRenderContext::Enable(IFX_FB_ALPHA_TEST).

Parameters:
eFunc Input IFXenum specifying the comparison function for alpha testing. The valid options are:
  • IFX_NEVER: The test never passes. No pixels are drawn.
  • IFX_LESS: The rendered alpha value must be less than the reference value.
  • IFX_GREATER: The rendered alpha value must be greater than the ref. value.
  • IFX_EQUAL: The rendered alpha value must be equal to the reference value.
  • IFX_NOT_EQUAL: The rendered alpha value must not be equal to the ref. value.
  • IFX_LEQUAL: The rendered alpha value must be less than or equal to the reference value.
  • IFX_GEQUAL: The rendered alpha value must be greater than or equal to the reference value.
  • IFX_ALWAYS: The test always passes. No pixels are rejected.
Returns:
An IFXRESULT error code.
Return values:
IFX_OK No error.
IFX_E_INVALID_RANGE The specified comparison function is invalid.
See also:
IFXRenderContext::Enable(), IFXRenderContext::Disable()
