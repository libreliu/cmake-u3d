Detailed Description

The IFXCLODManager class is used to change the resolution of an IFXMeshGroup.

It does this by adjusting the resolution on each IFXmesh in a synchronous manner so that cracks do not develop between materials. The resolution ranges from 0 to the number of vertices in the original mesh.

Note:
If a mesh has only one material, use the IFXResManagerInterface class to change resolution because it does away with overhead associated with keeping the resolution of each IFXMesh synchronized.
Warning:
The input mesh is assumed to be at full resolution.
