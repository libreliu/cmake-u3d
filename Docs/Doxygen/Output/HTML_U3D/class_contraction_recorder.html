)  [private]
 

This Method attempts to repair texture coordinates (and colored verts) in the updated face list so that textures don't jump much as a result of the contraction.

The method loops over all of the updated faces. The inner loop walks over all of the deleted faces. If the updated face has the same material ID as the deleted face, AND if those two faces had the same texture coodinate at the remove vertex, then the updated face will receive the texture coordinate of the deleted face at the keep vertex.
