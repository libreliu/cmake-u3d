IFXRESULT CIFXInterleavedData::Allocate (  U32  uNumVectors,
U32 *  puVectorSizes,
U32  uNumVertices
)  [virtual]
 

Allocates memory for the array.

The useable allocated memory begins on the start of a cache line. The array consists of a number of vertices. Each vertex has a number of vectors that comprise the vertex.

Parameters:
uNumVectors Input U32 specifying the number of separate vectors that make up each vertex.
puVectorSizes Input array of U32 values. Each entry in the array specifies the size in bytes of that corresponding vector entry in each vertex. The array MUST be at least uNumVectors in length, otherwise a memory corruption error can occur.
uNumVertices Input U32 specifying how many "vertices" the interleaved array should contain.
Returns:
An IFXRESULT value.
Return values:
IFX_OK No error.
IFX_E_OUT_OF_MEMORY There was a memory allocation error.
IFX_E_INVALID_POINTER The value specified for puVectorSizes was invalid (NULL).
IFX_E_INVALID_RANGE The value specified for was 0.

Implements IFXInterleavedData.

IFXRESULT CIFXInterleavedData::GetInfo (  U32 &  uNumVectors,
U32 *  puVectorSizes,
U32 &  uNumVertices
)  [virtual]
 

Gets information about the interleaved data array.

This method allows the user to determine the data arrangement of vectors within each vertex and the total number of vertices in the interleaved array. If puVectorSizes is NULL, the function will succeed and return other information. This will allow the correct array size to be allocated for a second call.

Parameters:
uNumVectors Reference to a U32 value that will be filled in with the number of vectors within each vertex of data.
puVectorSizes Pointer to an array of U32 values that will be filled in with vector sizes within each vertex. If non-NULL, this array must be at least uNumVectors in length.
uNumVertices Reference to a U32 value that will be filled with the number of vertices in the array.
Returns:
An IFXRESULT value.
Return values:
IFX_OK No error.

Implements IFXInterleavedData.

U32 CIFXInterleavedData::GetTimeStamp (   )  const [virtual]
 

Each instance of IFXInterleavedData can contain a user-defined U32 value referred to as a TimeStamp.

This method will retrieve it. It can be set with SetTimeStamp(). This method should NOT be used by anyone other than the rendering system. It is used to determine when static data is repeatedly rendered. This value is cleared to 0 when UpdateVersionWord() or Allocate() is called.
