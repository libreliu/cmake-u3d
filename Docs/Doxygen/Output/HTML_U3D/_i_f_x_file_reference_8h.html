enum IFXCollisionPolicy
 

Enumerator:
IFXCOLLISIONPOLICY_REPLACE  Replace existing object with the new object from external file.

Note:
This is not a reset because some existing items might not clash or overlap at all.
IFXCOLLISIONPOLICY_DISCARD  Discard the new object from external file.
IFXCOLLISIONPOLICY_PREPENDALL  Prepend scope name to object name for all objects from the external file.

Note:
The new name with prepended scope name may still collide with an existing object. In this situation, the new object from external file will replace that existing object.
IFXCOLLISIONPOLICY_PREPENDCOLLIDED  Prepend scope name to new obkect name if there is a collision.

Note:
The new name with prepended scope name may still collide with an existing object. In this situation, the new object from external file will replace that existing object.
IFXCOLLISIONPOLICY_POSTMANGLE  Append instance number to new object name if there is a collision.

Note:
Instance number will be unique.
