Detailed Description

This is the main interface for IFXTaskData.

Note:
The associated IID is named IID_IFXTaskData, and the CID for the default implementation is CID_IFXTaskData.

virtual IFXRESULT IFXAPI IFXTaskData::MatchPattern (  IFXTaskData *  pViewPattern,
BOOL *  pResult
)  [pure virtual]
 

This method compares this IFXTaskData instance to a reference instance.

The specific matching algorithm and success conditions are implementation defined. If the two instances are determined to match, pResult is set to TRUE; if they do not match, pResult is set to FALSE.

Parameters:
pViewPattern An interface pointer to another IFXTaskData, against which this instance will be compared. If this pointer is NULL, then the match is automatically considered to be a success.
pResult A pointer to a variable which will be set to TRUE or FALSE, according to the results of the match.
Returns:
Returns an IFXRESULT code.
  • IFX_OK upon success.
  • IFX_E_NOT_INITIALIZED if this object has not yet been initialized.
  • IFX_E_INVALID_POINTER if pViewPattern or pResult is NULL.
  • IFX_E_UNSUPPORTED if pViewPattern does not support a required interface.

Implemented in CIFXErrorInfo, CIFXNotificationInfo, CIFXSchedulerInfo, CIFXSimulationInfo, and CIFXTaskData.
